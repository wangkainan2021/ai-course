# 图片文件夹说明

请将5张图片放置在这个文件夹中，并命名为：

- `image1.jpg` - 第一张图片
- `image2.jpg` - 第二张图片
- `image3.jpg` - 第三张图片
- `image4.jpg` - 第四张图片
- `image5.jpg` - 第五张图片

## 支持的图片格式
- JPG / JPEG
- PNG
- GIF
- WebP

## 注意事项
- 图片文件名必须严格按照上述命名（image1.jpg 到 image5.jpg）
- 如果使用其他格式（如PNG），请在 `gallery.html` 中修改对应的文件扩展名
- 建议图片尺寸：宽度800-1200px，高度按比例
