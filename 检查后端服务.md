# 检查后端服务

## 当前状态
后端服务似乎没有成功启动。

## 手动启动步骤（请按顺序执行）

### 步骤1：打开命令提示符
- 按 `Win + R`
- 输入 `cmd`
- 按回车

### 步骤2：切换到后端目录
在命令提示符中输入：
```cmd
cd /d D:\课程设计\backend
```

### 步骤3：启动服务器
输入：
```cmd
node server.js
```

### 步骤4：查看输出
如果看到以下信息，说明启动成功：
```
后端服务器运行在 http://localhost:3000
API文档: http://localhost:3000/api
等待请求...
```

**重要**：保持这个窗口打开！

### 步骤5：测试
在浏览器中访问：
- http://localhost:3000/api/courses

应该看到：`{"success":true,"data":[]}`

## 如果启动失败

### 错误1：找不到模块
**解决方法**：
```cmd
cd /d D:\课程设计\backend
npm install
node server.js
```

### 错误2：端口已被占用
**解决方法**：修改端口号
1. 编辑 `backend/server.js`
2. 将 `const PORT = 3000;` 改为 `const PORT = 3001;`
3. 编辑前端文件，将API地址改为 `http://localhost:3001/api`

### 错误3：语法错误
查看命令提示符中的错误信息，告诉我具体的错误内容。

## 快速启动方式

我已经创建了以下启动文件：
- `启动后端.bat` - 在项目根目录
- `backend/start.bat` - 在后端目录

**双击运行** `backend/start.bat` 即可启动。

## 验证服务是否运行

启动后，在浏览器访问：
- http://localhost:3000/api/courses
- http://localhost:3000/api/levels

如果看到JSON数据，说明服务正常运行。
