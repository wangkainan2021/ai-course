# 测试后端连接

## 快速测试步骤

### 1. 检查后端服务是否运行

打开浏览器，访问：
- http://localhost:3000/api/courses
- http://localhost:3000/api/levels

如果看到JSON数据，说明后端正常运行。

### 2. 查看浏览器控制台

1. 打开管理端页面（frontend/admin.html）
2. 按 F12 打开开发者工具
3. 切换到 Console（控制台）标签
4. 尝试上传关卡或创建课程
5. 查看控制台中的日志信息

### 3. 常见错误及解决方法

#### 错误：Failed to fetch / Network error
**原因**：后端服务没有运行

**解决方法**：
```bash
cd backend
npm start
```

#### 错误：404 Not Found
**原因**：API路径错误或后端路由未配置

**解决方法**：检查API地址是否正确（http://localhost:3000/api）

#### 错误：500 Internal Server Error
**原因**：后端代码错误

**解决方法**：
1. 查看后端控制台的错误信息
2. 检查数据文件权限
3. 检查上传目录是否存在

### 4. 手动测试API

使用浏览器或Postman测试：

**创建课程**：
```
POST http://localhost:3000/api/courses
Content-Type: application/json

{
  "name": "测试课程",
  "description": "测试描述",
  "levelIds": []
}
```

**上传图片关卡**：
```
POST http://localhost:3000/api/levels/image
Content-Type: multipart/form-data

title: 测试关卡
description: 测试
images: [选择图片文件]
texts: ["说明1", "说明2"]
```
