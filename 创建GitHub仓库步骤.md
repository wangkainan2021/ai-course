## 步骤2：创建新仓库
1. 点击右上角的 **"+"** 号
2. 选择 **"New repository"**（新建仓库）

## 步骤3：填写仓库信息
- **Repository name（仓库名）**：`ai-course`
- **Description（描述）**：AI学习课程 - 小学三年级（可选）
- **Visibility（可见性）**：选择 **Public**（公开）
- ⚠️ **重要**：**不要**勾选以下选项：
  - ❌ Add a README file
  - ❌ Add .gitignore
  - ❌ Choose a license

## 步骤4：创建仓库
点击绿色的 **"Create repository"** 按钮

## 步骤5：完成
创建完成后，回到命令行执行推送命令即可。
