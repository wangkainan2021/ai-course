# GitHub部署指南

## 方式一：使用Git命令行（推荐）

### 1. 确保已安装Git
如果未安装，请从 https://git-scm.com/download/win 下载安装

### 2. 在项目目录中执行以下命令：

```bash
# 初始化Git仓库
git init

# 添加所有文件
git add .

# 提交文件
git commit -m "初始提交：AI学习课程项目"

# 添加远程仓库（替换为你的仓库名）
git remote add origin https://github.com/wangkainan2021/ai-course.git

# 推送到GitHub
git push -u origin main
```

如果遇到分支问题，可能需要：
```bash
git branch -M main
```

## 方式二：使用GitHub网页界面

### 1. 登录GitHub
访问 https://github.com 并使用你的账号登录

### 2. 创建新仓库
- 点击右上角的 "+" 号，选择 "New repository"
- 仓库名：`ai-course`（或你喜欢的名字）
- 选择 Public（公开）
- **不要**勾选 "Initialize this repository with a README"
- 点击 "Create repository"

### 3. 上传文件
- 在新建的仓库页面，点击 "uploading an existing file"
- 将项目中的所有文件拖拽到页面
- 在底部填写提交信息："初始提交：AI学习课程项目"
- 点击 "Commit changes"

## 启用GitHub Pages

### 1. 进入仓库设置
- 在仓库页面，点击 "Settings"（设置）

### 2. 配置Pages
- 在左侧菜单找到 "Pages"
- 在 "Source" 下选择 "main" 分支
- 选择 "/ (root)" 文件夹
- 点击 "Save"

### 3. 访问网站
几分钟后，你的网站将在以下地址可用：
`https://wangkainan2021.github.io/ai-course/`

（将 `ai-course` 替换为你的实际仓库名）
