# GitHub Pages 详细设置步骤

## 方法一：通过仓库设置页面

### 步骤1：进入仓库
访问：https://github.com/wangkainan2021/ai-course

### 步骤2：进入设置
- 点击仓库页面顶部的 **"Settings"**（设置）标签
- 注意：如果你看不到 Settings，可能是因为：
  - 你还没有登录
  - 你没有这个仓库的管理权限
  - 仓库是私有的，需要先公开

### 步骤3：找到Pages选项
在左侧菜单栏中，向下滚动找到：
- **"Pages"** 或 **"Pages settings"**
- 点击进入

### 步骤4：配置Pages
在Pages设置页面，你应该看到：

**"Build and deployment"（构建和部署）** 部分：
- **Source（源）**：点击下拉菜单
  - 选择 **"Deploy from a branch"**（从分支部署）
- **Branch（分支）**：选择 `main`
- **Folder（文件夹）**：选择 `/ (root)`
- 点击 **"Save"**（保存）

## 方法二：如果看不到Source选项

### 可能的原因：
1. **仓库是私有的**：GitHub Pages 免费版只支持公开仓库
   - 解决方法：将仓库改为公开
   - 步骤：Settings → 向下滚动到 "Danger Zone" → "Change repository visibility" → 改为 Public

2. **界面版本不同**：GitHub 有时会更新界面
   - 查找关键词："Build and deployment"、"Deploy from a branch"、"Branch"

3. **权限问题**：确保你是仓库的所有者或有管理员权限
